## Sobre

Portfolio profissional para psicologa clinica especializada em Terapia Cognitivo-Comportamental (TCC). Site com design elegante, tipografia serifada (Cormorant Garamond, Playfair Display) e experiencia de usuario acolhedora. Atendimento online e presencial em Sao Paulo.

---

## Tech Stack

### Frontend

| Tecnologia | Proposito |
|---|---|
| **React 18** | Biblioteca de interface |
| **JavaScript (JSX)** | Linguagem |
| **Vite 5** | Build tool e dev server |
| **Framer Motion** | Animacoes e transicoes |
| **React Router 7** | Roteamento SPA |
| **CSS puro** | Estilizacao |

### Infraestrutura

| Tecnologia | Proposito |
|---|---|
| **GitHub Pages** | Deploy estatico |
| **Google Analytics** | Monitoramento de visitas |
| **ESLint** | Lint e padronizacao |
| **Terser** | Minificacao de build |

---

## Funcionalidades

- **Design Elegante** — Tipografia serifada premium (Cormorant Garamond, Playfair Display, Nunito)
- **Animacoes Suaves** — Transicoes e micro-interacoes com Framer Motion
- **100% Responsivo** — Experiencia acolhedora em qualquer dispositivo
- **SEO Otimizado** — Meta tags e descricoes para mecanismos de busca
- **Google Analytics** — Monitoramento de trafego
